# EventBusClient Documentation

This folder contains documentation for the EventBusClient library.

## Structure

```
docs/
├── README.md           # This file
├── adr/                # Architecture Decision Records
│   └── README.md       # ADR index and template
└── diagrams/           # Architecture and sequence diagrams
    └── README.md       # Diagram index
```

## Contents

### ADR (Architecture Decision Records)

The `adr/` folder contains Architecture Decision Records documenting significant design decisions made during development.

See [adr/README.md](adr/README.md) for the ADR index and template.

### Diagrams

The `diagrams/` folder contains architecture diagrams, sequence diagrams, and other visual documentation.

See [diagrams/README.md](diagrams/README.md) for the diagram index.
